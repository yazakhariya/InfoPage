# Info Page

## Stack:
- React
- Redux
- TypeScript
- Styled Components

## Для начала работы необходимо:
1. Сохранить проект локально (save the project locally): 
```bash
git clone https://github.com/yazakhariya/InfoPage.git
```  
2. Ввести в консоли команду (type the command):
```bash 
npm run dev
```
